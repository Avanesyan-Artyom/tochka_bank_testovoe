## Описание задачи
Требуется предварительно отфильтровать веб-тексты для претрейна LLM, обучив модель бинарно классифицировать тексты по трём критериям качества:

- **Integrity** — целостность и единство темы  
- **Factuality** — наличие фактов о мире  
- **Truthfulness** — отсутствие противоречий common-sense знаниям  

Для каждого критерия решается **задача бинарной классификации (0 / 1)**.  
Примеры с меткой **0.5** считаются неопределёнными и **не участвуют в обучении и подсчёте метрики**.

## Данные
Используются предоставленные файлы:
- `train.parquet`
- `test.parquet`

Фичи:
- `text` — основной текст
- `*_reasoning` — reasoning модели (используется только на train)

## Пайплайн решения

### 1. Подготовка данных
- Загружаются train и test датасеты  
- Примеры, где целевая метка равна `0.5`, **исключаются**  
- Данные из train разбиваются на `train / validation` со стратификацией  

### 2. Работа с дисбалансом классов
- В данных наблюдается выраженный дисбаланс между классами `0` и `1`
- Для его компенсации используется **взвешенный Binary Cross-Entropy**
- Значение `pos_weight` рассчитывается отдельно для каждого критерия на основе распределения классов в train-выборке
- Это позволяет штрафовать ошибки по редкому классу сильнее и стабилизировать обучение

### 3. Формирование датасета
- Используется токенизатор `DeepPavlov/rubert-base-cased`
- Для **train** в текст добавляется reasoning (`*_reasoning`)
- Для **validation и test** используется только исходный текст
- Для каждого примера формируются:
  - `input_ids`
  - `attention_mask`
  - метки `integrity`, `factuality`, `truthfulness`

### 4. Модель
- Реализована **мультитасковая модель** на базе `ruBERT`
- Общий encoder (`rubert-base-cased`)
- Три независимые классификационные головы — по одной на каждый критерий
- Обучение проводится совместно по всем задачам

### 5. Обучение и оценка
- Используется взвешенный `CrossEntropyLoss`
- Основная метрика — **F1-score**
- Примеры с истинной меткой `0.5` игнорируются при подсчёте
- Итоговая метрика — среднее арифметическое F1 по трём критериям

## Результаты

**F1-score на валидации:** **0.9493**
